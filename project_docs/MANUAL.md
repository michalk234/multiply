# MANUAL PRACY Z PROJEKTEM

Jak pracować:

1. Zawsze zaczynaj od:
- PROJECT_STATE.md
- WORKFLOW_RULES.md

2. Do zmian:
- wysyłaj tylko potrzebny plik lub fragment

3. Gdy robimy większą zmianę:
- wyślij cały projekt

4. Nigdy:
- nie wysyłaj wszystkiego bez potrzeby

Cel:
- szybka praca
- małe iteracje
