# DOCUMENTATION

This document describes WHAT the application is and HOW it works.

## Project
Multiplication Table SPA

## Goal
Help children practice multiplication in a simple, fast way.

## Structure (Current)
- app.js → bootstrap
- model.js → logic
- view.js → UI
- controller.js → flow

## Flow
1. Select difficulty
2. Generate task
3. Enter answer
4. Validate
5. Show feedback
6. Next task

## Testing
- Model is tested
- Coverage available via Vitest

## Notes
- No backend
- No persistence
- Mobile-first
