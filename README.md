# Tabliczka mnożenia

Prosta aplikacja SPA do nauki tabliczki mnożenia dla dzieci.

## Struktura plików
- `index.html`
- `styles.css`
- `app.js`
- `README.md`

## Co jest w tej wersji
- wybór poziomu: `30 / 50 / 100`
- 10 pytań w rundzie
- pasek postępu
- licznik pytania i poprawnych odpowiedzi
- feedback inline bez alertów
- błędna odpowiedź przekreślona + pokazanie poprawnej
- wynik końcowy: liczba poprawnych, procent, czas, poziom
- pilnowanie focusu w polu odpowiedzi, żeby nie trzeba było klikać
- przycisk `OK` do zatwierdzania odpowiedzi na iPhone i innych urządzeniach mobilnych
- brak duplikatów pytań w jednej rundzie

## Uruchomienie lokalnie
Otwórz `index.html` w przeglądarce.

## GitHub Pages
1. Wrzuć wszystkie pliki do repozytorium GitHub.
2. Wejdź w `Settings` → `Pages`.
3. Ustaw publikację z gałęzi `main` i folderu `/root`.
4. Zapisz ustawienia i poczekaj na link do strony.
